第6课程笔记

1.JavaScript由ECMAScript、DOM和BOM三部分组成。

​ Javascript规范是由ECMA组织制订的；DOM规范是由W3C组织制订的；BOM没有规范。

2.BOM模型对象结构

​ window
​ |- document - DOM
​ |- history
​ |- location
​ |- navigator
​ |- screen
​ |- frames[]

3.Window对象是BOM与DOM的顶层对象，表示浏览器窗口或者网页中的框架。

4.setTimeout与setInterval的区别：（面试题（笔试））
​ 1）相同点：
​ 都是计时器（定时器），主要用来实现动画。
​ 2）不同点：
​ a)setTimeout只触发一次；而setInterval会持续触发。
​ b)setTimeout是在时间到了之后再触发；而setInterval是在时间到来之前触发。
5.DOM：文档对象模型。
​ 是一个使程序和脚本能够动态地访问和更新文档的内容、结构以及样式，并独立于平台和语言的接口。
​ DOM可以用来操作XML和HTML，以及CSS等。
6.HTML DOM中访问HTML元素方法
​ 1）获取body对象
​ document.body
​ 2）ID
​ document.getElementById(‘id名’); // 返回一个DOM对象
​ 3）class
​ document.getElementsByClassName(‘类名’); // 返回一个DOM对象数组
​ 4）标签
​ document.getElementsByTagName(‘标签名’); // 返回一个DOM对象数组
​ 5）name属性的元素
​ document.getElementsByName(‘name值’); // 返回一个DOM对象数组

以上获取DOM可以用在所有浏览器上，不需要处理兼容性。

7.获取DOM的另外写法：
​ 1）document.querySelector()
​ a.id
​ document.querySelector(‘#id’)
​ b.class
​ document.querySelector(‘.cls’)
​ c.tagname
​ document.querySelector(‘tagname’)
​ 2）document.querySelectorAll():获取当前文档中所有DOM节点

以上方法，IE9以下不支持，所以要慎用！

8.获取DOM元素小结
​ 1）通过ID获取
​ document.getElementById() 返回一个值
​ 2）通过Class获取
​ document.getElementsByClassName() 返回一个数组
​ 3）通过标签名获取
​ document.getElementsByTagName() 返回一个数组
​ 4）通过name值获取
​ document.getElementsByName() 返回一个数组
​ 5）获取表单组件
​ document.forms[下标].name值
​ 或：
​ document.表单name值.name值
​ 6）获取body
​ document.body 返回一个值
​ 7）获取html
​ document.documentElement 返回一个值
​ 8）通过选择器获取一个元素
​ document.querySelector() 返回一个值
​ 9）通过选择器获取一组元素
​ document.querySelectorAll() 返回一个数组
9.阻止默认形为（面试点）
​ var e = e || window.event;
​ if(e.preventDefault){
​ e.preventDefault();
​ }else{
​ e.returnValue = false;
​ }
10.阻止事件冒泡（面试点）（从内层向外层触发）（元素有包含关系，并且绑定相同事件触发时）
​ var e = e || window.event;
​ if(e.stopPropagation){
​ e.stopPropagation();
​ }else{
​ e.cancelBubble = true;
​ }
11.事件捕获（从外层向内层触发）（面试点）

12.利用hexo框架创建一个个人博客
​ 1）安装node.js（因为要用node.js中的npm（包管理器），利用npm可以安装你想要的包文件）
​ 可以在官方下载：http://nodejs.cn/download/

    Tips:
        安装node.js时，最好选用默认路径，不要修改路径，否则需要设置环境变量。

2）安装完node.js后，在要创建项目的地方打开命令窗口，输入：
    node -v
    (如果显示的有node的版本号，说明你的node.js安装成功，否则要重新安装，或设置环境变量）
3）安装 hexo（最好不要用cnpm安装）脚手架（脚手架是用来搭建项目环境的）
      npm install --global hexo-cli    （install：安装，--global：全局安装）
      或：
      npm i -g hexo-cli
4）创建项目
    hexo init 项目名        （init:初始化一个项目，项目名不要写用中文命名）
5）启动项目
    a）cd 项目名
    b）生成静态页面至public目录
        hexo generate
        或：
            hexo g
    c）开启预览访问端口
        hexo server        （默认端口号为4000）
        或：
            hexo s
        或：
            hexo s -p 端口号
    d）在浏览器地址栏中输入：
        localhost:4000

    Tips:
        如果修改了配置文件或项目内容，需要先清除public目录
            hexo clean
        再重新生成public目录
            hexo g
        最后启动服务
            hexo s

        以上三步可能要频繁使用。
6）将博客部署到github上
    a）打开github.com网站，并注册，然后登录
        提示：github是全球的一个用来托管用户代码的网站。
    2）创建一个仓库，仓库地址格式如下：
        您登录仓库的用户名.github.io
    3）安装插件
        npm i hexo-deployer-git --save
        或：
        npm i hexo-deployer-git --save --registry=https://registry.npm.taobao.org
    4）修改项目中的配置文件_config.yml，在该文件最后添加如下代码：
        deploy:
          type: git
          repo: https://github.com/您登录仓库的用户名/您登录仓库的用户名.github.io.git
          branch: master
    5）将项目推送到github上。
        hexo d
