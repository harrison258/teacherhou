第6课程笔记

1.JavaScript由ECMAScript、DOM和BOM三部分组成。

​ Javascript规范是由ECMA组织制订的；DOM规范是由W3C组织制订的；BOM没有规范。

2.BOM模型对象结构

​ window
​ |- document - DOM
​ |- history
​ |- location
​ |- navigator
​ |- screen
​ |- frames[]

3.Window对象是BOM与DOM的顶层对象，表示浏览器窗口或者网页中的框架。

4.setTimeout与setInterval的区别：（面试题（笔试））
​ 1）相同点：
​ 都是计时器（定时器），主要用来实现动画。
​ 2）不同点：
​ a)setTimeout只触发一次；而setInterval会持续触发。
​ b)setTimeout是在时间到了之后再触发；而setInterval是在时间到来之前触发。
5.DOM：文档对象模型。
​ 是一个使程序和脚本能够动态地访问和更新文档的内容、结构以及样式，并独立于平台和语言的接口。
​ DOM可以用来操作XML和HTML，以及CSS等。
6.HTML DOM中访问HTML元素方法
​ 1）获取body对象
​ document.body
​ 2）ID
​ document.getElementById(‘id名’); // 返回一个DOM对象
​ 3）class
​ document.getElementsByClassName(‘类名’); // 返回一个DOM对象数组
​ 4）标签
​ document.getElementsByTagName(‘标签名’); // 返回一个DOM对象数组
​ 5）name属性的元素
​ document.getElementsByName(‘name值’); // 返回一个DOM对象数组

以上获取DOM可以用在所有浏览器上，不需要处理兼容性。

7.获取DOM的另外写法：
​ 1）document.querySelector()
​ a.id
​ document.querySelector(‘#id’)
​ b.class
​ document.querySelector(‘.cls’)
​ c.tagname
​ document.querySelector(‘tagname’)
​ 2）document.querySelectorAll():获取当前文档中所有DOM节点

以上方法，IE9以下不支持，所以要慎用！

8.获取DOM元素小结
​ 1）通过ID获取
​ document.getElementById() 返回一个值
​ 2）通过Class获取
​ document.getElementsByClassName() 返回一个数组
​ 3）通过标签名获取
​ document.getElementsByTagName() 返回一个数组
​ 4）通过name值获取
​ document.getElementsByName() 返回一个数组
​ 5）获取表单组件
​ document.forms[下标].name值
​ 或：
​ document.表单name值.name值
​ 6）获取body
​ document.body 返回一个值
​ 7）获取html
​ document.documentElement 返回一个值
​ 8）通过选择器获取一个元素
​ document.querySelector() 返回一个值
​ 9）通过选择器获取一组元素
​ document.querySelectorAll() 返回一个数组
9.阻止默认形为（面试点）
​ var e = e || window.event;
​ if(e.preventDefault){
​ e.preventDefault();
​ }else{
​ e.returnValue = false;
​ }
10.阻止事件冒泡（面试点）（从内层向外层触发）（元素有包含关系，并且绑定相同事件触发时）
​ var e = e || window.event;
​ if(e.stopPropagation){
​ e.stopPropagation();
​ }else{
​ e.cancelBubble = true;
​ }
11.事件捕获（从外层向内层触发）（面试点）
​ 从顶层元素到目标元素或者，和事件冒泡是一个相反的过程。

父级DOM.addEventListener('事件名', function () {
    ……
},true);

12.利用hexo框架创建一个个人博客
​ 1）安装node.js（因为要用node.js中的npm（包管理器），利用npm可以安装你想要的包文件）
​ 可以在官方下载：http://nodejs.cn/download/

    Tips:
        安装node.js时，最好选用默认路径，不要修改路径，否则需要设置环境变量。

2）安装完node.js后，在要创建项目的地方打开命令窗口，输入：
    node -v
    (如果显示的有node的版本号，说明你的node.js安装成功，否则要重新安装，或设置环境变量）
3）安装 hexo（最好不要用cnpm安装）脚手架（脚手架是用来搭建项目环境的）
      npm install --global hexo-cli    （install：安装，--global：全局安装）
      或：
      npm i -g hexo-cli
4）创建项目
    hexo init 项目名        （init:初始化一个项目，项目名不要写用中文命名）
5）启动项目
    a）cd 项目名
    b）生成静态页面至public目录
        hexo generate
        或：
            hexo g
    c）开启预览访问端口
        hexo server        （默认端口号为4000）
        或：
            hexo s
        或：
            hexo s -p 端口号
    d）在浏览器地址栏中输入：
        localhost:4000

    Tips:
        如果修改了配置文件或项目内容，需要先清除public目录
            hexo clean
        再重新生成public目录
            hexo g
        最后启动服务
            hexo s

        以上三步可能要频繁使用。
6）将博客部署到github上
    a）打开github.com网站，并注册，然后登录
        提示：github是全球的一个用来托管用户代码的网站。
    2）创建一个仓库，仓库地址格式如下：
        您登录仓库的用户名.github.io
    3）安装插件
        npm i hexo-deployer-git --save
        或：
        npm i hexo-deployer-git --save --registry=https://registry.npm.taobao.org
    4）修改项目中的配置文件_config.yml，在该文件最后添加如下代码：
        deploy:
          type: git
          repo: https://github.com/您登录仓库的用户名/您登录仓库的用户名.github.io.git
          branch: master
    5）将项目推送到github上。
        hexo d

第5天课程笔记

1.对象
​ 对象是类的实例化，但es5中没类的概念。在es5中用构造函数实现类的定义。
​ 对象是由属性和方法组成的，它属于引用类型。
2.对象分类
​ 内置对象和用户自定义对象
​ 内置对象：String/Math/Date/Function….
3.基本数据类型与引用数据类型的区别
​ 基本数据类型：number string boolean undefined null
​ 引用数据类型: object
​ 区别：
​ 　　基本类型和引用类型，都是数据类型；
​ 　　数据的值都存储在内存中；
​ 　　程序在运行时，会把内存划分不同区域，其中最重要的两个区域，是栈和堆。
​ 　　栈中的值占用空间固定，而堆中是动态分配的内存；
​ 　　所有的变量都存储在栈
​ 　　基本数据类型的值直接存储在栈中，而引用类型的值存储在堆中，变量只是一个指向堆内存的地址引用。
4.用户自定义对象定义并赋值
​ 1）new
​ 方法1：
​ var obj = new Object();
​ obj.name = ‘张三’
​ obj.age = 18
​ obj.fn = function(){
​ …
​ }
​ 方法2（不推荐）：
​ var obj = new Object({name:’Tom’,tel:’1379080900’});
​ 2）字面量
​ 方法1：
​ var obj = {}
​ obj.name = ‘张三’
​ obj.age = 18
​ obj.fn = function(){
​ …
​ }
​ 方法2：
​ var obj ={
​ name:’Tom’,
​ tel:’1379080900’,
​ fn: function(){
​ …
​ }
​ };

注意：
    对象的属性和方法是以键值对的形式呈现，中间用冒号隔开，以逗号结束，最后一项不用加逗号。

5.String对象
​ 用来生成一个字符串。
​ 用法：
​ new String()
​ 或：
​ new String(‘abc’)
6.字符串属性
​ length:字符串长度
7.字符串常用方法
​ 1）charAt(下标):返回指定下标所对应的字符。如果下标越界，则返回空字符串
​ 2）indexOf(字符串[,下标]):返回指定字符串所在下标；如果有第二个参数，则表示从指定位置查找指定的字符串，找到返回下标，没找到返回空串
​ 3）lastIndexOf(字符串[,下标]):返回一个指定的字符串值最后出现的位置，在一个字符串中的指定位置从后向前搜索
​ 4）replace(字符串1或正则表达式，字符串2):字符串中用字符串2替换字符串1或按正则要求替换
​ 5）substr(下标[,长度]):从指定下标开始截取指定长度的字符。如果没长度，表示从指定下标开始截取到最后
​ 6）substring(下标1[,下标2]):从下标1截取到下标2之前，不包含下标2的内容，如果没下标2，表示从指定下标开始截取到最后
​ 7）toLowerCase():把字符串转换为小写
​ 8）toUpperCase():把字符串转换为大写
​ 9）charCodeAt(下标):返回在指定的位置的字符的 Unicode 编码。
​ 10）fromCharCode(数值列表):将数值（ Unicode 编码）转换为所对应的字符。
8.Number对象
​ 用来定义一个数值对象。
​ 方法：
​ toFixed(小数位数):把数字转换为字符串，结果的小数点后有指定位数的数字。
9.Math对象
​ 使用Math对象的属性和方法时，前面的Math不能省略。
​ 1）属性
​ PI
​ 2）方法
​ abs：求绝对值
​ pow：幂函数
​ sqrt：平方根函数
​ round：四舍五入取整函数
​ ceil：向上取整函数（天花板函数）(取一个大于或等于原数的一个最小整数）
​ floor：向下取整函数（地板函数）(取一个小于或等于原数的一个最大整数）
​ random：随机函数（返回[0,1)之间的随机小数）
​ Math.random(n):中的n表示随机种子。
​ min：取最小值函数
​ max：取最大值函数
10.Date对象
​ 方法：
​ getFullYear()
